# Nexlink dApp Documentation

Developer documentation for building dApps on the Nexlink platform.

Nexlink is a mobile wallet and social app. Third-party dApps run inside the Nexlink in-app browser (with automatic identity injection) or in external browsers (with QR code login). This documentation covers the authentication and integration protocols.
